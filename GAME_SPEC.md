# Night Shift: Mall Security

## Pitch
Co-op horror-comedy where players patrol a procedurally generated mall, detect anomalies, fix them, and file reports before instability reaches 100%.

## Target
Steam (Windows first), $5–$7, low-poly/flat shading, short chaotic runs (10–20 min), 2–6 players later.

## Core Loop
Patrol → Detect anomaly → Fix anomaly → File report → Escalation.

## Win / Lose
Win: survive until 6:00 AM.
Lose: Instability reaches 100% (or all players down later in co-op).

## Constraints
- Favor systems over polish.
- No multiplayer until Phase 2.
- No custom shaders, minimal post-processing.
